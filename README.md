## Contribute
- Fork the repository
- clone the ssh code into your local workspace.
- pick an issue you want to work on and let us know by a comment.
- create a branch, git checkout -b <branch-name>
- git add .
- git commit -m 'your commit'
- git push origin <branch-name>
- Make pull request
